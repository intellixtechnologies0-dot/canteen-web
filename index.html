How it works

  1. New orders arrive in the queue with a token, table number, items, and ETA.
  2. Tap Start to begin preparation; timer turns active.
  3. Mark Ready when complete; counters update instantly.
  4. Orders turn Delayed if they cross ETA; get alerted to prioritize.

This demo runs locally with mock data and no backend. Hook it up to your API or Firebase to go live.
